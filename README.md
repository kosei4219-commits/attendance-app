# 📱 出退勤打刻アプリ

研修生用の出退勤管理PWAアプリケーションです。シンプルな操作で出勤・退勤の記録、勤務時間の自動計算、課題完了報告ができます。

## ✨ 主な機能

- **🏢 出勤打刻** - ワンクリックで出勤時刻を記録
- **🏠 退勤打刻** - 退勤時刻を記録し、勤務時間を自動計算
- **🎯 課題完了報告** - 完成したアプリのURLを簡単に報告
- **📊 Google Spreadsheet連携** - すべてのデータを自動保存
- **💾 ローカルストレージ** - オフラインでも動作

## 🚀 使い方

### 出勤打刻
1. アプリを開く
2. 「🏢 出勤打刻」ボタンをクリック
3. 出勤時刻が記録され、スプレッドシートに保存されます

### 退勤打刻
1. 「🏠 退勤打刻」ボタンをクリック
2. 退勤時刻が記録され、勤務時間が自動計算されます
3. スプレッドシートに保存されます

### 課題完了報告
1. 「🎯 課題完了報告」ボタンをクリック
2. 完成したアプリのURLを入力
3. 「送信」ボタンで報告完了

## 🛠 技術スタック

### フロントエンド
- **HTML5** - セマンティックマークアップ
- **CSS3** - モダンなグラデーションデザイン、レスポンシブ対応
- **JavaScript (ES6+)** - 非同期処理、localStorage管理

### バックエンド・連携
- **Google Apps Script** - スプレッドシート操作のサーバーレスバックエンド
- **Google Spreadsheet** - データストレージ
- **LINE Messaging API** - グループ通知（オプション）

### ホスティング
- **GitHub Pages** - 静的サイトホスティング（HTTPS対応）

## 📱 レスポンシブデザイン

- **スマートフォン最適化** - 1画面で全機能にアクセス可能
- **タッチ操作対応** - 指で押しやすい44x44px以上のボタン
- **デスクトップ対応** - 最大幅480pxで中央配置
- **クロスブラウザ** - Chrome、Safari、Edge、Firefox対応

## 🎨 デザイン特徴

- グラデーション背景とカード型UI
- スムーズなアニメーション
- モダンなモーダルUI
- 視認性の高いステータス表示

## 📋 データ構造

### Google Spreadsheet
3つのシートで構成：

1. **研修生マスタ** - ユーザー情報
2. **打刻記録** - 日次の出退勤データ
3. **課題完了記録** - 提出したアプリURL

### ローカルストレージ
- ユーザー情報（研修生ID、氏名）
- 当日の打刻情報（出勤時刻、退勤時刻、勤務時間）

## 🔒 セキュリティ

- HTTPS通信（GitHub Pages標準対応）
- APIキーはサーバー側（Google Apps Script）で管理
- クライアント側には秘密情報を保存しない
